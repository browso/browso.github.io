# Chat Commands

Commands are optional shortcuts. Type `/` in the AI panel to see suggestions.
Normal questions do not need a command.

## `/help`

Shows the commands available in your version of Browso.

## `/save [note]`

Saves the current page for future questions. The note is optional.

```text
/save Useful source for warranty comparisons
```

Clicking the bookmark button in the AI panel also saves the page, without a
note. Saving the same page again updates it instead of creating another copy.

## `/notes`

Lists your ten most recently updated saved pages, including their addresses and
notes.

## `@something to remember`

Saves a short preference or instruction directly to memory:

```text
@Prefer concise answers with source links
```

Memory must be enabled in **Settings > Memory**.

## Keyboard Tips

- `Enter` sends.
- `Shift+Enter` adds a new line.
- `Ctrl+E` or `Cmd+E` shows or hides the AI panel.

Browso selects the appropriate agent mode automatically. `/mode` is not a
supported command.
